RetroVerse Updater.app
======================

A macOS application bundle for launching the RetroVerse Data Updater GUI.

USAGE:
------
Double-click "RetroVerse Updater.app" to launch the updater GUI.

ADD TO DOCK:
------------
1. Open Finder and navigate to this location
2. Drag "RetroVerse Updater.app" to your Dock
3. The app will appear in your Dock and can be launched from there

The app automatically:
- Changes to the correct project directory
- Launches the GUI using system Python
- Handles all relative paths correctly
